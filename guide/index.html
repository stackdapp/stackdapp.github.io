Quick Start Guide

Up and running in 10 minutes

Stackd is set up by parents, used by kids. Here's exactly how to get started — from first launch to your kid's first XP.

Step 1 — Parent setup

Set everything up behind the PIN

The first thing to do is get into the Parent dashboard and customize the app for your kid. Your default PIN is 1234 — change it once you're in.

Tap the Parent button in the top-right corner of any screen

1

Set your kid's name

Go to Parent → Settings. Enter your kid's name — it shows up in greetings throughout the app.

2

Change your PIN

Still in Settings, set a 4-digit PIN only you know. This locks the parent dashboard so kids can't change their own rewards.

3

Build the schedule

Go to Parent → Schedule. Add blocks for the main parts of your kid's day — Wake Up, Breakfast, School, Free Time, Bedtime. Each block gets a time and XP value.

Tip: Start with 4–6 blocks. Too many at once is overwhelming. You can always add more after the first week.
4

Add tasks

Go to Parent → Tasks. Add the daily chores and habits you want your kid to complete — "Make bed", "Brush teeth", "Feed the dog". Each task earns XP.

5

Set up rewards

Go to Parent → Rewards. Add things your kid actually wants — 30 minutes of screen time, choosing dinner, staying up late. Set an XP cost for each.

Tip: Set costs high enough that rewards feel earned but not impossible. 100–300 XP is a good starting range for daily rewards.
6

Tap Save Changes

The purple bar at the top of the Parent dashboard shows your unsaved changes. Tap Save Changes when you're done — nothing is saved until you do.

Step 2 — Kid intro

Hand it over and walk through day one together

The first time, sit with your kid and show them how it works. Once they see the XP counter go up, they get it fast.

🏠

Home screen

Shows the current schedule block, XP earned today, level, and streak. The big number is their score.

✅

Tasks tab

Tap a task to check it off. XP adds up instantly. Unchecking is allowed — accidents happen.

⭐

Rewards tab

Shows their XP balance and available rewards. They tap to spend — you confirm with your PIN.

📚

School tab

Tap Start Session to begin. A timer counts down each subject with a gentle warning before time runs out.

Common questions

Quick answers

Will my kid's data be erased if I clear the app cache?
No. All data is stored in your browser's localStorage, which is separate from the app cache. Clearing the cache only updates the app — your schedule, tasks, XP, and rewards stay safe.
Can my kid change their own rewards or schedule?
No. The Parent dashboard requires your PIN every time. Kids can see rewards and spend XP, but can't edit anything without PIN confirmation.
Does it work offline?
Yes. Once loaded, Stackd works fully offline. Save it to your home screen for the best experience — it runs like a native app.
How do I save it to the home screen?
On iPhone: tap the Share button in Safari, then "Add to Home Screen". On Android: tap the three-dot menu in Chrome, then "Add to Home Screen" or "Install app".
What's the difference between the 3 app versions?
Full — schedule, tasks, school timer, rewards, and report all in one. School — school timer and XP only, designed for homeschool days. Schedule — schedule and tasks only, no school timer.
How do I backup my setup?
Go to Parent → Settings → Download Backup. This saves a JSON file with your entire setup. You can restore it anytime from the same screen.
