Welcome to Stackd
Stackd helps kids manage their day with schedules, tasks, timers, and rewards - all in one place.
This guide takes about 2 minutes. You can skip it anytime.
🔒
Set up as a parent first
Tap the Parent button in the top right. Your default PIN is 1234 - change it in Parent → Settings.
From the Parent dashboard you can set your kid's name, customize the schedule, add chores, and create rewards.
🗓
Build your kid's day
Go to Parent → Schedule to add blocks like "Wake Up", "School", "Free Time". Each block has a time, XP value, and optional sub-items.
Start with 4–6 blocks for the first week. You can always add more later.
⭐
XP and rewards
Kids earn XP by completing tasks and blocks. They spend it in the Rewards tab on things you set up - like screen time, a snack, or staying up late.
Set reward costs high enough that they feel earned. Start with 3–4 rewards to keep it simple.
🚀
You're ready to go
Hand the phone to your kid and walk through the first day together. The routine builds quickly once they see the XP adding up.
You can revisit this guide anytime from Parent → Settings → Quick Start Guide.
